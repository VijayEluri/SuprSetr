Managing Your Sets

To allow SuprSetr to manage one of your sets, double click the set and you will be taken to the Set Editor window. Once you check the box that says "Manage this set with SuprSetr", you can define the parameters of the set. Parameters include:

  • Tags
  • Date Taken
  • Date Uploaded
  • Sorting

Once you define the parameters of the set, SuprSetr will know how to search your Flickr stream for photos that match the parameters, and can then update the content of the photoset based on the search results. SuprSetr will add and remove photos from the set to make it match the results of the search. However, SuprSetr will NEVER delete photos from your photostream.

Twitter

If you authorize SuprSetr to use your Twitter account, you can choose to send a tweet each time new photos are added to your sets. You can enable and disable tweets for each photoset, and you can customize the message per set. When customizing the message, you can specify some tokens that will be filled in:

  • %t - This will be replaced with the photoset title
  • %u - This will be replaced with the photoset URL. The URL will be shortened with bit.ly
  • %c - This will be replaced with the number of photos that were added to the photoset
  • %C - This will be replaced with the total number of photos in the photoset

When making a custom tweet template for a set, make sure to keep in mind the 140 character limit, remembering that the token replacement will most likely make the message longer than it appears in the text box. SuprSetr will trim the final message to fit within 140 characters, truncating the message to fit.
